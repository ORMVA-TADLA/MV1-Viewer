# MV1 Data Viewer

A simple web application for viewing and filtering data from the MV1 dataset. Built with Streamlit and Pandas, this tool allows you to interactively explore your CSV data, view summary metrics, and filter by key columns.

## Features

-   📊 Interactive data table for the `MV1.csv` file
-   Summary metrics (e.g., total records)
-   Filter data by columns: Matricule, Code Client, Code Parcelle, Tertiaire
-   User-friendly web interface

## Usage
Access the MV1 Data Viewer [Live](https://ormvat-td.streamlit.app/)
